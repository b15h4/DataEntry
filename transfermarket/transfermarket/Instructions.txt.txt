Install Python3 -- https://www.python.org/downloads/

Install Scrapy -- pip install scrapy

Troubleshooting Scrapy :
1 . If C++ error appears when installing scrapy , go install Tools for Visual Studio
2 . Link -- https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2017
3 . Run again

How To Run Script :

1 . Run cmd
2 . Go to folder transfermarket where the items, settings, pipelines appear -- cd /transfermarket/transfermarket  or 2 times cd /transfermarket -- Tested in windows
3 . scrapy crawl transferspy
4 . Output will appear in same follder your in

How to add more links :

1. Go to transfermarket folder , open links.txt file and you can add the links , as many as you want
2. Save the file
3. Delete previous output
4. Run Script again
